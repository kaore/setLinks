setLinks is a function that takes in a string and will wrap any link it finds with an 'a' anchor as such: http://cibul.net

It can take in a className option (like this: www.worlds-worst-website.com) in case you feel like appending classes to your link

  • A wrapper function exists as well if you want (http://google.com) to just apply setLinks on a list of elements.
  • The function works on emails, bogusemail@fakedomain.com, anotherfake@email.com
  • It takes in a target option if you need to set it to "_blank" (little url here: http://www.youtube.com/watch?v=EmLHOGT0v4c)
  • It should work with emails containing subsubdomains like this one bogusemail@fake.domain.com
